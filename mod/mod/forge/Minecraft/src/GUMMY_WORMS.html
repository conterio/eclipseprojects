Now your Gummy_Worm class file should look like this:

package Items;

import net.minecraft.item.ItemFood;
import net.minecraft.creativetab.CreativeTabs;
import net.minecraft.potion.Potion;
import net.minecraftforge.fml.common.registry.GameRegistry;

public class Gummy_Worm extends ItemFood {

public Gummy_Worm(int amount, boolean isWolfFood) {
super(amount, isWolfFood);
GameRegistry.registerItem(this,"gummyworm");
this.setUnlocalizedName("gummyworm");
this.setCreativeTab(CreativeTabs.tabFood);
this.setPotionEffect(Potion.moveSpeed.id,6,5,1.0F);
this.setAlwaysEdible();

}

}

Now we need create the .json file telling Minecraft how to display our item.

In eclipse click on the folder 'src/main/resources'.

Right click on the package called 'assets.examplemod.models.item' and select New > File.
If file isn't there, select New > Other. Then type the word 'file' and click next.

Name the file gummyworm.json

Paste the following code in gummyworm.json

{ "parent": "builtin/generated", "textures": { "layer0": "examplemod:items/gummy_worm" }, "display": { "thirdperson": { "rotation": [ -90, 0, 0 ], "translation": [ 0, 1, -3 ], "scale": [ 0.55, 0.55, 0.55 ] }, "firstperson": { "rotation": [ 0, -135, 25 ], "translation": [ 0, 4, 2 ], "scale": [ 1.7, 1.7, 1.7 ] } } }

There is only one line you need to verify. Look at the line (4th from the top) that says:

"layer0": "examplemod:items/gummy_worm"

gummy_worm needs to match the name of your gummy worm image.

So if your image is named gummy_worm.png then it should say gummy_worm.

Now lets copy your gummy_worm image to the correct package.

Under the folder 'src/main/resources' find the package called 'assets.examplemod.textures.items'

Move your gummy_worm.png into that package.

Last you need to name your item so Minecraft knows what to call it.

Under the folder 'src/main/resources' open the package called 'assets.examplemod.lang'

Double click on the file called US_eng.lang

Add the following code to that file. Click on the line for an explanation.

item.gummyworm.name=Gummy Worm item.gummyworm.name - this is what Minecraft know the name of our item as.
Gummy Worm - You can put anything you want here. Whatever you type will be displayed as the name of the item in the game.
